# docker
Concentrá la infraestructura del proyecto, dejando espacio para los Dockerfile e imágenes que
se requieran. Contiene archivos, configuraciones, y demás.

## Servicios
- `app`: Lógica de negocio principal
- `scraper`: Scraper de páginas web dinámicas
- `airflow` (futuro): Orquestador de tareas ETL
