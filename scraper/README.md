# scraper
Este directorio fue creado con el propósito de aislar el scraping de páginas web dinámicas, con 
su propio contenedor y scripts que no forman parte del flujo natural, si no más bien para obtener
información que no es necesario actualizar de manera frecuente.

**¿Por qué tomé esta decisión?** El scraping de páginas web dinámicas es pesado, y aún más los contenedores
que hacen estas tareas posibles, y sería un proceso que relentizaría todo el flujo, por lo que cambiar la
frecuencia de actualización me pareció la manera más práctica de solucionar este problema.
